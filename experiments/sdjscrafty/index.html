What is Crafty.js

Per craftyjs.com, Crafty is "Crafty is a JavaScript game library that can help you create games in a structured way..."

Features

  • An Intuitive way to structure your game ("Entity/Component")
  • DOM/Canvas rendering baked in
  • Sprite/Animation/Tween Stuff
  • Lots of prebaked awesome

Crafty.e("2D, DOM, Color, Collision")
    .color('rgb(0,0,255)')
    .attr({ x: 300, y: 150, w: 10, h: 10,
            dX: Crafty.math.randomInt(2, 5),
            dY: Crafty.math.randomInt(2, 5) })

        //hit floor or roof
        if (this.y <= 0 || this.y >= 290)

            this.x = 300;
            Crafty("RightPoints").each(function () {
                this.text(++this.points + " Points") });
        }

        this.x += this.dX;
        this.y += this.dY;
    })
    .onHit('Paddle', function () {
    this.dX *= -1;
})

Entities

Entities are things in the game, e.g. a player, a tree, a bullet, a platform.


// Create a new entity, and associate it to the following components (comma separated)
Crafty.e("Paddle, 2D, DOM, Color, Multiway")
// Then call "color" (added in the Color component)
    .color('rgb(255,0,0)')
// Add dimensions to it (added in the "2D" component)
    .attr({ x: 20, y: 100, w: 10, h: 100 })
// Make it playable (added in the Multiway component)
    .multiway(4, { W: -90, S: 90 });

Components

Components let you add bits of functionality to different entities in structural ways. You can build on top of other components by calling "requies"


// Define a new component, Thing
Crafty.c('Thing', {
    // In the init function
    init: function(){
        //  Extend the 2D, Dom rendering ,and Color components
        this.requires('2D, DOM, Color');
        // Then call the color method
        this.color('rgb(255,0,255)');
        // And a part of the 2d method
        this.attr({w:30,h:30});
            
    },
});
